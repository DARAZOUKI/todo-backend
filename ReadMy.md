## üìå API Endpoints

| Method   | Endpoint      | Description          | 
|----------|--------------|-----------------------| 
| **GET**  | `/todos`     | Get all todos         | 
| **POST** | `/todos`     | Add a new todo        | 
| **PUT**  | `/todos/:id` | Update a todo by ID   | 
| **DELETE** | `/todos/:id` | Delete a todo by ID |
